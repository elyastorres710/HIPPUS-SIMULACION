# Iteración 2 - Modelo Matemático (ejemplo)

Esta carpeta contiene los scripts para la segunda iteración del proyecto, enfocada en la generación de datos considerando el modelo matemático del sistema vestibular-pupilar.

## Scripts

- `generacion_datos.py` - Generación de datos basados en modelo matemático
- `analisis.py` - Análisis de datos con modelo real
- `clasificacion.py` - Clasificación con datos realistas
- `evaluacion.py` - Evaluación comparativa con iteración 1

## Ejecución

```bash
# Ejecutar la iteración completa
python generacion_datos.py
python analisis.py
python clasificacion.py
python evaluacion.py
```

## Características

- **Datos**: Señales basadas en modelo matemático del sistema vestibular-pupilar
- **Enfoque**: Datos realistas y biológicamente plausibles
- **Objetivo**: Validar el sistema con datos que representen la realidad

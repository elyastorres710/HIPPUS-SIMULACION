# Directivas del Proyecto

El siguiente documento contiene las directivas para el desarrollo del proyecto. los pasos que se deben seguir para el desarrollo iterativo, y las reglas que se deben seguir para la creacion de funciones.

## Objetivo del Proyecto
Desarrollo de un sistema de clasificación automática de migraña vestibular utilizando aprendizaje de máquina explicables, basado en el análisis del hippus pupilar. Utilizando para esto simulaciones de modelos matemáticos de la dinámica del sistema vestibular-pupilar.
